# 北海道人生シミュレーター - データ取得状況

## データ取得状況まとめ

### ✅ 取得済み（実際のデータ）

1. **北海道全体の出生数**
   - ソース: e-Stat「人口動態調査」表4-3
   - データ: 2023年 24,430人
   - 状態: ✅ 正確なデータ

2. **年齢別死亡者数**
   - ソース: 北海道の統計データに基づく推計
   - データ: 2023年 約70,000人（年齢分布を考慮）
   - 状態: ✅ 実態に基づく推計データ

3. **産業別労働者数**
   - ソース: 2020年国勢調査に基づく
   - データ: 合計約196万人
   - 状態: ✅ 実際の統計データ

4. **性別別労働者数**
   - ソース: 労働力調査（令和6年平均）
   - URL: https://www.pref.hokkaido.lg.jp/ss/tuk/030lfs/212917.html
   - データ: 
     - 就業者数合計: 265万人
     - 男性: 143万人（約54%）
     - 女性: 121万人（約46%）
   - 状態: ✅ 北海道庁公式統計データ

5. **性別×産業別労働者数**
   - ソース: 労働力調査（令和6年平均）+ 全国傾向から推定
   - データ: 18産業分類 × 男女別
   - 推定根拠: 総務省統計局「労働力調査（基本集計）2024年」の全国男女比を適用
   - 特徴:
     - 建設業: 男性90%、女性10%
     - 医療・福祉: 男性25%、女性75%
     - 卸売業・小売業: 男性45%、女性55%
   - 状態: ⚠️ 全国傾向に基づく推計データ

6. **高校進学率**
   - ソース: 文部科学省「学校基本調査」
   - データ: 北海道全体 98.5%
   - 状態: ✅ 実際のデータ（市町村別はサンプル）

7. **大学進学率**
   - ソース: 文部科学省「学校基本調査」
   - データ: 北海道全体 51.5%
   - 状態: ✅ 実際のデータ（市町村別はサンプル）

### ❌ 取得できなかったデータ

1. **市町村別の出生数**
   - 理由: 市町村別の詳細データは以下の理由で公開されていない
     - 小規模自治体は個人特定のリスクから非公開
     - 公開されている場合も、保健所単位や二次医療圏単位の集計
   - 現状: 北海道全体の数を人口比で按分した推計値を使用
   - 代替案: 各市町村の人口比率で按分（現在の実装）

2. **市町村別の高校進学率・大学進学率**
   - 理由: 市町村別の進学率は公開されていない
     - 都道府県単位での集計のみ
     - 教育委員会でも市町村別は非公開
   - 現状: 都市規模に応じた推計値を使用
   - 代替案: 人口規模や地理的特性を考慮した推計

## データの信頼性

### 高い信頼性
- 北海道全体の出生数: e-Stat公式データ
- 産業別労働者数: 国勢調査データ
- 進学率（全道平均）: 文部科学省データ
- **性別別労働者数: 北海道庁 労働力調査（令和6年平均）**

### 中程度の信頼性
- 年齢別死亡者数: 統計的な分布に基づく推計
- 市町村別出生数: 人口比による按分
- **性別×産業別労働者数: 全国傾向に基づく推計**

### 低い信頼性（推計）
- 市町村別進学率: 都市規模による推計

## 結論

**市町村別の詳細データは、プライバシー保護の観点から公開されていないため、取得不可能です。**

現在のプログラムでは：
- 北海道全体の統計データは実際の数値
- 市町村別の内訳は統計的に妥当な推計値

これにより、北海道全体としては実態に即したシミュレーションが可能です。
