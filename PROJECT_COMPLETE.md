## 🎯 3つのバージョン

### 1. Webアプリ - シンプル版
**ファイル**: `app.py`  
**起動方法**: `./start.sh`

**特徴**:
- ブラウザで操作可能
- 美しいUI
- 1-20人の人生を生成
- 詳細データの展開表示
- モバイル対応
- **初心者におすすめ！**

### 2. Webアプリ - 拡張版
**ファイル**: `app_advanced.py`  
**起動方法**: `./start_advanced.sh`

**特徴**:
- シンプル版の全機能
- 統計分析機能
- グラフ表示（5種類）
  - 出生地分布
  - 産業分布（円グラフ）
  - 死因分布
  - 大学進学先分布
  - 年齢分布（ヒストグラム）
- 10-1000人の大量生成
- **データ分析におすすめ！**

### 3. コマンドライン版
**ファイル**: `hokkaido_life_simulator.py`  
**起動方法**: `python3 hokkaido_life_simulator.py -n 10`

**特徴**:
- テキストベース
- スクリプトで自動化可能
- 無制限の人数生成
- **自動化・バッチ処理におすすめ！**

## 🌐 デプロイオプション

2. Streamlit Cloudに連携
3. `app.py`を指定してデプロイ

→ 数分で世界中からアクセス可能！

1. `Procfile`と`setup.sh`が既に用意済み
2. `heroku create`
3. `git push heroku main`

→ 本格的なWebアプリとして公開可能！
