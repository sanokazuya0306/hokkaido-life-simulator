## 🎯 今すぐできること

### 1. ブラウザでアクセス
- Chrome、Safari、Firefoxなどのブラウザを開く
- アドレスバーに `http://localhost:8501` を入力
- Enterキーを押す

### 2. 人生を生成
- 画面中央の「🎲 人生を生成する」ボタンをクリック
- 北海道の統計データに基づいた人生が表示されます

### 3. 設定を変更
- 画面左側のサイドバーで設定変更
  - 生成人数：1〜20人
  - シード値：再現性のある結果

### 4. 詳細を確認
- 「📋 詳細データを見る」をクリック
- 出生地、進学、就職、死因などの詳細が表示されます

---

## 📱 モバイルでも使える

### スマホやタブレットから

1. スマホを同じWi-Fiに接続
2. ブラウザで以下を開く：
   ```
   http://192.168.1.6:8501
   ```
3. シンプル版はモバイル対応しているので快適に使えます

## 🎨 拡張版に切り替える

統計分析とグラフ表示機能が欲しい場合：

1. **現在のアプリを停止**
   - ターミナルで `Ctrl + C`

2. **拡張版を起動**
   ```bash
   ./start_advanced.sh
   ```
   または
   ```bash
   python3 -m streamlit run app_advanced.py
   ```

### 拡張版の追加機能
- ✅ 統計分析（進学率、平均寿命など）
- ✅ 出生地分布グラフ
- ✅ 産業分布（円グラフ）
- ✅ 死因分布グラフ
- ✅ 大学進学先分布
- ✅ 年齢分布（ヒストグラム）
- ✅ 10〜1000人の大量生成

## 💡 Tips

### ページをリロードする
- ブラウザで F5 または Command+R（Mac）

- 画面左上の「>」アイコンをクリック

- 画面右上の「⋮」メニュー → 「Settings」→ 「Wide mode」

- 画面右上の「⋮」メニュー → 「Settings」→ 「Theme」→ 「Dark」

## ⚠️ トラブルシューティング

### ページが開かない
- URLが正しいか確認：`http://localhost:8501`
- Webアプリが起動しているか確認

### エラーが出る
- ターミナルのエラーメッセージを確認
- `python3 check_setup.py` でセットアップを確認

### 動作が遅い
- 生成人数を減らす
- ブラウザのキャッシュをクリア
- 他のタブを閉じる
