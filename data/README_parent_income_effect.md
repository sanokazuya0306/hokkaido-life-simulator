# 親の世帯年収が子の学歴に与える影響データ

## 概要

このデータファイル（`parent_income_effect.csv`）は、親の世帯年収が子の進学率に与える影響を補正係数として定義しています。

**重要**: このデータは複数の研究・統計から導出した**推定値**であり、特定の論文から直接引用した数値ではありません。

## 補正係数の根拠

### 主要な参照データ

#### 1. 文部科学省「21世紀出生児縦断調査」分析

貧困経験の有無による大学進学率の差異が確認されています：

| 貧困経験区分 | 大学進学率 |
|-------------|-----------|
| 非貧困持続群 | 63.4% |
| 貧困脱出群 | 43.0% |
| 貧困突入群 | 39.1% |
| 貧困持続群 | 35.4% |

出典: 文部科学省「21世紀出生児縦断調査」に基づく分析
参考: https://www.pacola.co.jp/子ども期の貧困が大学進学に及ぼす影響

この比率（63.4% / 35.4% ≒ 1.79）から、世帯年収による進学率の差は約1.8倍程度と推定されます。

#### 2. 高等教育の修学支援新制度の効果分析

- **準対象世帯（世帯年収210万〜370万円以下）**: 制度開始前後で大学等への進学率が約61.5%から約70.7%に向上
- 低所得層への支援が進学率向上に効果があることを示唆

出典: 大学ジャーナルオンライン（2024年）
https://univ-journal.jp/215904/

#### 3. 東京大学「学生生活実態調査」（2023年度・第72回）

東大生の家庭の世帯年収分布：

| 世帯年収 | 割合 |
|---------|------|
| 450万円未満 | 9.4% |
| 750万円〜950万円未満 | 12.3% |
| 950万円〜1050万円未満 | 9.3% |
| 1050万円〜1250万円未満 | 12.5% |
| 1250万円以上 | 20.4% |
| 分からない | 26.4% |

950万円以上の家庭が全体の42.2%を占めており、全国平均世帯年収（約536万円）と比較して高所得層が多い。

出典: 東京大学学生生活実態調査（2023年度）
参考: https://financial-field.com/income/entry-433353/

#### 4. 国民生活基礎調査（2024年）

- 全国平均世帯年収: 約536万円
- この値を基準（補正係数1.0）として設定

出典: 厚生労働省「国民生活基礎調査」（2024年）

#### 5. 高校ランクの媒介効果に関する研究

日本の長期パネル調査を使った研究で、家族所得が大学進学に与える影響の約25%が「高校の選抜性（ランク）」を経由することが確認されています。

出典: PubMed（2024年11月）
https://pubmed.ncbi.nlm.nih.gov/39542611/

### 補正係数の算出方法

| 年収階級 | 高校進学補正 | 大学進学補正 | 算出根拠 |
|---------|-------------|-------------|----------|
| 100万円未満 | 0.92 | 0.55 | 貧困持続群の進学率35.4%を基準に推定 |
| 100〜200万円 | 0.94 | 0.60 | 貧困経験群の平均的な進学率を反映 |
| 200〜300万円 | 0.96 | 0.70 | 準対象世帯の進学率約61.5%を参考 |
| 300〜400万円 | 0.98 | 0.80 | 中間層下位の進学率を推定 |
| 400〜500万円 | 1.00 | 0.90 | 全国平均年収付近 |
| 500〜700万円 | 1.00 | 1.00 | 基準値（全国平均付近） |
| 700〜1000万円 | 1.00 | 1.10 | 高所得層の進学率上昇を反映 |
| 1000〜1500万円 | 1.00 | 1.20 | 東大生の42%が950万円以上の家庭出身 |
| 1500万円以上 | 1.00 | 1.30 | 最高所得層の教育投資効果を反映 |

### 補正の適用方法

シミュレーションでは、世帯年収と親学歴の両方の補正係数を組み合わせて適用します：

```
調整後進学率 = 地域別基準進学率 × 親学歴補正係数 × 世帯年収補正係数
```

ただし、親学歴と世帯年収には相関があるため、補正が過度にならないよう調整しています。

## 注意事項

1. **推定値である**: 「世帯年収別の大学進学率」の直接的な公的統計は日本では限定的であり、関連研究から推定した値です。

2. **保守的な設定**: 実際の格差はより大きい可能性がありますが、シミュレーションでは地域別進学率・親学歴との組み合わせで過度な偏りを避けるため、より保守的な補正係数を採用しています。

3. **複合的な要因**: 実際には世帯年収だけでなく、親の学歴、居住地域、教育への関心、塾・習い事への投資など複合的な要因が進学率に影響します。

4. **時代変化**: 高等教育の修学支援新制度（2020年〜）などにより、低所得層の進学率は改善傾向にあります。

5. **地域差**: 首都圏と地方では教育機会や選択肢に差があり、同じ年収でも進学率に違いが生じる可能性があります。
