# 親学歴が子学歴に与える影響データ

## 概要

このデータファイル（`parent_education_effect.csv`）は、親の最終学歴が子の進学率に与える影響を補正係数として定義しています。

**重要**: このデータは複数の研究・統計から導出した**推定値**であり、特定の論文から直接引用した数値ではありません。

## 補正係数の根拠

### 主要な参照データ

#### 1. OECD Education at a Glance 2025（日本のデータ）

- **親が高等教育（大学等）修了の場合**: 25〜34歳の子の高等教育修了率は **約72%**
- **親が上位中等教育（高校等）修了の場合**: 25〜34歳の子の高等教育修了率は **約43%**

出典: OECD (2025), Education at a Glance 2025: OECD Indicators, Japan Country Note
https://www.oecd.org/en/publications/education-at-a-glance-2025_1a3543e2-en/japan_8f0a8541-en.html

この比率（72% / 43% ≒ 1.67）から、親が大学卒の場合は高卒の場合と比較して約1.7倍の進学率補正が妥当と推定されます。ただし、シミュレーションでは地域別進学率との組み合わせで使用するため、より保守的な値（1.2倍）を採用しています。

#### 2. ベネッセ教育総合研究所「子どもの生活と学びに関する親子調査」

- **父母ともに大学卒業の家庭**: 高校1年の夏までに大学進学希望を形成する割合は **約52%**
- **父母ともに非大学卒の家庭**: 同割合は **約25%**

出典: ベネッセ教育総合研究所（2020年代調査）
https://benesse.jp/berd/special/childedu_researcher/yamaguchi.html

#### 3. 文部科学省 全国学力・学習状況調査（学力テスト）

親の学歴を含む社会経済的背景（SES）と学力の関係:
- SESが高い家庭の子どもは、低い家庭と比較して正答率が高い傾向
- 算数・数学では約18ポイント以上の差が確認されている

出典: 文部科学省「全国学力・学習状況調査」経年変化分析
https://www.mext.go.jp/a_menu/shotou/gakuryoku-chousa/

#### 4. お茶の水女子大学「保護者に対する調査の結果と学力等との関係の専門的な分析」（平成29年度）

- SES（家庭所得、父母の学歴等を合成）を四分位に分けた分析
- 高SES群と低SES群で学力に有意な差

出典: 文部科学省委託研究（平成29年度）
https://www.mext.go.jp/a_menu/shotou/gakuryoku-chousa/1406895.htm

#### 5. SSM調査（社会階層と社会移動全国調査）

- 親の学歴が子の教育達成に影響を与える傾向は一貫して確認されている
- 若年コーホートでは親学歴に由来する格差が拡大傾向

出典: 日本社会学会 SSM調査シリーズ
https://www.jstage.jst.go.jp/article/jsr/59/4/59_4_682/_article/-char/ja

### 補正係数の算出方法

| 親学歴 | 高校進学補正 | 大学進学補正 | 算出根拠 |
|--------|-------------|-------------|----------|
| 中学校 | 0.95 | 0.60 | OECD: 親が高等教育未修了の場合の修了率43%を基準に、さらに低い層を推定 |
| 高校 | 1.00 | 0.80 | 基準値（OECD: 親が上位中等教育修了の場合43%） |
| 短大・専門学校 | 1.00 | 1.00 | 高等教育修了に含まれるが、大学卒より低めに設定 |
| 大学 | 1.00 | 1.20 | OECD: 親が高等教育修了の場合72%（43%の約1.67倍）を保守的に1.2倍に調整 |
| 大学院 | 1.00 | 1.30 | 大学卒よりさらに高い教育投資傾向を反映 |

### 補正の適用方法

シミュレーションでは、両親の学歴から補正係数の平均を取り、地域別の基準進学率に乗じます：

```
調整後進学率 = 地域別基準進学率 × (父親補正係数 + 母親補正係数) / 2
```

## 注意事項

1. **推定値である**: 直接的な「親学歴別子の進学率」の公的統計は日本では限定的であり、関連研究から推定した値です。

2. **保守的な設定**: OECDデータでは親学歴による差は約1.7倍ですが、地域別進学率との組み合わせで過度な偏りを避けるため、より保守的な補正係数を採用しています。

3. **複合的な要因**: 実際には親の学歴だけでなく、世帯収入、居住地域、教育への関心など複合的な要因が進学率に影響します。

4. **時代変化**: 教育機会の拡大により、親学歴の影響は時代とともに変化する可能性があります。
