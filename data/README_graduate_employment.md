# 北海道の大学卒業者就職状況データ

## データソース

**北海道労働局「新規大学等卒業者の就職状況」**
- URL: https://jsite.mhlw.go.jp/hokkaido-roudoukyoku/news/2025/0530_00001.html
- PDF: https://jsite.mhlw.go.jp/hokkaido-roudoukyoku/content/contents/002368226.pdf
- 調査時点: 各年3月31日現在
- 調査対象: 道内の全大学、短期大学、高等専門学校、専修学校

## ファイル

### `hokkaido_graduate_employment.csv`

北海道の大学等卒業者の道内・道外就職状況データ

**列の説明:**
- `卒業年度`: 卒業年月（例: 2025年3月）
- `学校種別`: 大学、短期大学、高等専門学校、専修学校、総計
- `卒業者数`: 卒業者総数
- `求職者数`: 就職を希望する人数
- `就職者数`: 実際に就職した人数
- `道内就職者数`: 北海道内に就職した人数
- `道外就職者数`: 北海道外に就職した人数
- `就職率`: 求職者に対する就職者の割合（%）
- `道内就職割合`: 就職者のうち道内就職の割合（%）
- `道外就職割合`: 就職者のうち道外就職の割合（%）

## 主要な発見（2025年3月卒）

### 大学
- **就職者数**: 14,685人
- **道内就職**: 8,936人 (60.9%)
- **道外就職**: 5,749人 (39.1%)
- **就職率**: 94.6%（前年比+1.5ポイント）

### 学校種別による道内残留率の違い

| 学校種別 | 道内就職割合 | 道外就職割合 |
|---------|-------------|-------------|
| 短期大学 | 94.1% | 5.9% |
| 専修学校 | 82.0% | 18.0% |
| **大学** | **60.9%** | **39.1%** |
| 高等専門学校 | 26.8% | 73.2% |

**特徴:**
- 短期大学・専修学校は道内就職が大半
- **大学は約4割が道外に就職**
- 高等専門学校は7割以上が道外就職（技術者として本州企業に就職）

## 進学データとの比較

### 高校卒業後の大学進学（2024年度）
- **道内大学進学**: 73.54%
- **道外大学進学**: 26.46%

### 大学卒業後の就職（2025年3月卒）
- **道内就職**: 60.9%
- **道外就職**: 39.1%

### 考察
1. 高校生の約7割は道内大学に進学
2. しかし大学卒業者の約4割は道外に就職
3. **「道内進学→道外就職」という人材流出パターンが存在**
4. 道内大学で学んだ人材が卒業後に道外へ流出している

## 時系列での変化

| 年度 | 道内就職割合 | 道外就職割合 |
|------|-------------|-------------|
| 2019年3月 | 68.6% | 31.4% |
| 2020年3月 | 66.6% | 33.4% |
| 2021年3月 | 70.2% | 29.8% |
| 2022年3月 | 68.4% | 31.6% |
| 2023年3月 | 67.9% | 32.1% |
| 2024年3月 | 66.4% | 33.6% |
| **2025年3月** | **68.0%** | **32.0%** |

**トレンド:**
- 2024年は道外就職が33.6%と増加傾向だったが、2025年は32.0%に減少
- 全体として、道内就職は65-70%、道外就職は30-35%で推移

## データの更新方法

北海道労働局は毎年5月末頃に前年度3月卒業生のデータを公表します。

1. 北海道労働局のウェブサイトをチェック: https://jsite.mhlw.go.jp/hokkaido-roudoukyoku/
2. 「新規大学等卒業者の就職状況」のプレスリリースを探す
3. PDFから最新データを取得
4. CSVファイルに追記

## 参考情報

### 北海道労働局について
- 厚生労働省の地方支分部局
- 北海道内の労働行政を担当
- 雇用対策、労働基準、職業安定などの業務を実施

### 調査方法の注意点
- 北海道労働局の調査: 道内全大学等を対象、3月31日時点
- 厚生労働省・文部科学省の全国調査: 抽出調査、4月1日時点
- 調査方法が異なるため、全国データとは直接比較できない場合がある
