### 🚀 起動方法

#### 1️⃣ Webアプリ（シンプル版）- おすすめ！
```bash
./start.sh
```
- 初心者向け
- 美しいUI
- 手軽に試せる

#### 2️⃣ Webアプリ（拡張版）- データ分析向け
```bash
./start_advanced.sh
```
- 統計分析機能
- グラフ表示
- 大量生成対応

#### 3️⃣ コマンドライン版 - 自動化向け
```bash

```
- スクリプト実行

- テキスト出力

### 🎯 こんな使い方ができます

#### 初めて使う方
```bash
./start.sh
```
→ ブラウザでボタンをクリックするだけ！

#### データを分析したい方
```bash
./start_advanced.sh
```
→ 100人分のデータを生成してグラフで可視化！

#### スクリプトで使いたい方
```bash
python3 hokkaido_life_simulator.py -n 100 > results.txt
```
→ 結果をテキストファイルに保存！

### 💡 特徴

- ✅ 北海道庁の公式統計データを使用
- ✅ 188市町村の出生データ
- ✅ リアルな進学率・就職率
- ✅ 統計に基づいた死亡年齢・死因
- ✅ 美しいWebインターフェース
- ✅ モバイル対応

### 📊 使用データ

すべて**北海道庁が公開している公式統計データ**：

1. 市町村別出生数（188市町村、2024年）
2. 高校進学率（178市町村、2024年度）
3. 大学進学率（125市町村、2024年度）
4. 大学進学先（48都道府県、2024年度）
5. 産業別労働者数（18産業、2024年）
6. 定年年齢分布（2022年）
7. 年齢別死亡者数（2022年）
8. 死因別死亡者数（2022年）

### 🆘 トラブルシューティング

#### エラーが出る場合
```bash

python3 check_setup.py

# 2. ライブラリを再インストール
pip3 install -r requirements.txt

# 3. もう一度セットアップを確認
python3 check_setup.py
```

#### ポートが使用中の場合
```bash
# 別のポートで起動
streamlit run app.py --server.port=8502
```
